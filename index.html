Seshstation Installation Guide

It is important to follow the installation steps in correct order otherwise you may brick your device. Clean flash is recommended

  • Wipe Dalvik, Cache, System and Data
  • Flash ROM, Gapps (if needed)
  • Flash DFE (optional), Flash Magisk (Optional But Recommended)
  • Reboot To Recovery, Flash Seshstation ZIP

After successful completion of the above steps, you can now boot your device. Enjoy The Power Of Seshstation

Thermals Installation Guide

If you want to overclock your device CPU, GPU and want to get fast charging then you have to install the custom thermals to disable thermal throtlling.

  • Open Magisk
  • Select the thermal ZIP (Ben's Thermal) Recommended
  • Reboot Your Device
  • Enjoy beast performance and fast-charging

You can use any thermal you want, but it is recommended to use the Ben's Thermal v6

Fast-Charging Usage Guide

In order to achieve Fast-Charging on your device, Thermals are required to be installed on your device.

  • Install any kernel manager app with script support such as Smart Kernel Manager or Franco Kernel Manager
  • Select the fast-charging script
  • Allow the script to be executed on boot
  • Enjoy the fast-charging speed upto 3300 mA

The default charging speed of POCO F1 is set at 2800 mA. Using this fast-charging script you can achieve speeds upto 3300 mA

Frequently Asked Questions
(FAQ)

What to do if device is overheating?

If you have issues with heat than uninstall or don't install thermals and you'll have normal temps but remember without thermals you won't be able to have the same charging speeds and performance will drop a bit.

How to get more CPU performance?

If you want even more CPU performance, go to Tips from Franco Manager and disable CPU underclock.

What to do if screen is stuck?

If you screen gets stuck, and you have LKT, then uninstall and try again. If your screen gets stuck right after you closed it, and touch doesn't respond, use lower hz.

How to fix PUBG crash?

Well it seems like after the new PUBG update... PUBG has introduced anti-root mechanism in the APP. So it will crash if you have Magisk or any other Super User App installed ... This affects the users from PUBG version 0.17... Simply follow the steps and it will be fine. Only works on Magisk as of now...
[FIX] PUBG crashes after 0.17 update
first install magisk
-go to magisk and open magisk setting
-click on hide magisk manager
-close app & then restart magisk
-it will download again. after download close app again & then restart
-go to again magisk setting
-click hide apps
-hide pubg
-close app
-reboot device & enjoy the game
Basically Hide PUBG in Magisk.. Pubg started checking for root and it simply crashes just to avoid the hackers using scripts. Same is with iPhone and Jailbreak. It doesn't work on iPhone until you hide the jailbreak (like Magisk hide) with a tweak. Credits : XDA
